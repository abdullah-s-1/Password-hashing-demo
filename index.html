How It Was Made

This demo uses plain HTML for structure, CSS for layout and feedback, and vanilla JavaScript for the interactive logic. Real SHA digests are generated with the browser's built-in Web Crypto API.

  • HTML and CSS handle the layout, cards, indicators, and FAQ section.
  • Vanilla JavaScript updates the strength meter and swaps hashing modes.
  • Web Crypto API handles the SHA-256 and SHA-1 digest operations.
  • crypto.getRandomValues() generates a new random salt in the browser.
